#+title: jtx1inst

Basic C library for accessing two Jetson TX1's INA3221 -- current and voltage monitors as well as on-module temperature sensors -- either through sysf nodes and userspace I2C.

* Build and install
#+begin_src sh
mkdir build
cd build
cmake ..
make
sudo make install
sudo ldconfig
#+end_src sh
